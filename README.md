# NoCommandPrefixes
A way to remove the ugly fallback prefix on commands.

## Author
+ [MrZCookie](https://gitlab.com/mrzcookie)

## Config

```
# Interval to check for a command with prefix.
Interval: 180
# Disable all commands with a prefix.
Disable-All-Prefixes: true
# Prefixes to disable.
Prefixes:
  - minecraft
  - bukkit
```
